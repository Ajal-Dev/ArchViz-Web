ArchViz

Architectural visualization, commonly referred to as "archviz," is the practice of creating visual representations of architectural designs. These representations can range from simple sketches and drawings to highly detailed and photorealistic 3D models and animations. Archviz is used by architects, designers, real estate developers, and marketing teams to visualize buildings and interior spaces before they are constructed.

House

2 BHK House:

This is the overview model of the 2 BHK house model. It is a 2 floor house which has a pool as well. The player has multiple accessibility feature, including free roam which is the walk mode to see and inspect the model and the Edit mode which allows the player to choose the user preferred assets and materials.

Walk Mode:

This is the preview of the player Walk Mode which allows the player to free roam the model and inspect the assets and materials present in the model. The model has a interactable door feature which can be used whle the player is in walk mode. It depicts the visual representation of the model and allows the user to judge the model based on a visual aspect.

Edit Mode:

This is the preview of the edit mode for the House Model. As we can see, the player has access to the edit menu where they can change multiple assets and materials for those respective assets. The edit mode allows the user to change various assets such as the beds, kitchen facility, dining table, house walls, and the floor materials of the 1st floor, ground floor as well as the balcony.
